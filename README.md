# Functionality
- Adding crypto assets
- Displaying important info about coins/crypto portfolio in different formats(chart/table)
- Getting current information about chosen coin

# Used technologies
-  JavaScript
-  React
-  react-chartjs-2
-  AntD
-  HTML
-  CSS
